# –û—Ç—á–µ—Ç –ø–æ –ª–∞–±–æ—Ä–∞—Ç–æ—Ä–Ω–æ–π —Ä–∞–±–æ—Ç–µ ‚Ññ2 ¬´–†–µ—à–µ–Ω–∏–µ –Ω–µ–ª–∏–Ω–µ–π–Ω—ã—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏–π¬ª
## –ó–∞–¥–∞—á–∞ 2.1
–ú–µ—Ç–æ–¥–æ–º –ø—Ä–æ—Å—Ç–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ –Ω–∞–π—Ç–∏  –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ –∫–æ—Ä–Ω–∏ –Ω–µ–ª–∏–Ω–µ–π–Ω–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è  $f(ùë•)=0$ —Å —Ç–æ—á–Ω–æ—Å—Ç—å—é $\varepsilon = 10^{-8}$.

$f(x) = x^3 - 0.9x^2 - x - 0.1$.

### –†–µ—à–µ–Ω–∏–µ
–ü–æ—Å—Ç—Ä–æ–∏–º –≥—Ä–∞—Ñ–∏–∫ —Ñ—É–Ω–∫—Ü–∏–∏ –∏ –æ–ø—Ä–µ–¥–µ–ª–∏–º –æ—Ç—Ä–µ–∑–∫–∏ –ª–æ–∫–∞–ª–∏–∑–∞—Ü–∏–∏ –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –∫–æ—Ä–Ω—è:![[University/sem4/–ß–ú4/Lab2/211.png]]

–û–ø—Ä–µ–¥–µ–ª–∏–º –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é $f(x)$:
$f'(x) = 3x^2 - 1.8 x - 1$.
–ü–æ—Å—Ç—Ä–æ–∏–º –≥—Ä–∞—Ñ–∏–∫ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–π –∏ –æ—Ç–º–µ—Ç–∏–º –Ω–∞ –Ω—ë–º –≥—Ä–∞–Ω–∏—Ü—ã –æ—Ç—Ä–µ–∑–∫–æ–≤ –ª–æ–∫–∞–ª–∏–∑–∞—Ü–∏–∏:![[University/sem4/–ß–ú4/Lab2/212.png]]
–ò–∑ –≥—Ä–∞—Ñ–∏–∫–∞ –≤–∏–¥–Ω–æ, —á—Ç–æ –Ω–∞ –æ—Ç—Ä–µ–∑–∫–∞—Ö –ª–æ–∫–∞–ª–∏–∑–∞—Ü–∏–∏ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏–∏ —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–π –∑–Ω–∞–∫.

–î–ª—è –∫–∞–∂–¥–æ–≥–æ –∫–æ—Ä–Ω—è –æ–ø—Ä–µ–¥–µ–ª–∏–º –∏—Ç–µ—Ä–∞—Ü–∏–æ–Ω–Ω—ã–π –ø–∞—Ä–∞–º–µ—Ç—Ä $\alpha$ –∏ –ø–∞—Ä–∞–º–µ—Ç—Ä $q$, –∏—Å–ø–æ–ª—å–∑—É—è —Ñ–æ—Ä–º—É–ª—ã:
$\alpha = \dfrac{2}{M1 + m1}$,
$q = \left| \dfrac{M1 - m1}{M1 + m1} \right|$,
–≥–¥–µ $M1=\max_\limits{x\in [a,b]} f'(x),\ m1 = \min_\limits{x\in [a,b]}f'(x)$.

–°–æ—Å—Ç–∞–≤–∏–º –ø—Ä–æ–≥—Ä–∞–º–º—É –¥–ª—è –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è –∫–æ—Ä–Ω—è —Å –∑–∞–¥–∞–Ω–Ω–æ–π —Ç–æ—á–Ω–æ—Å—Ç—å—é $\varepsilon$ –ø–æ –º–µ—Ç–æ–¥—É –ø—Ä–æ—Å—Ç—ã—Ö –∏—Ç–µ—Ä–∞—Ü–∏–π. –í –∫–∞—á–µ—Å—Ç–≤–µ  —Ä–∞—Å—á–µ—Ç–Ω–æ–π —Ñ–æ—Ä–º—É–ª—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º –º–µ—Ç–æ–¥ –ø—Ä–æ—Å—Ç–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ —Å –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–º:
$x_{n + 1} = x_n - \alpha f(x_n)$
```python
def MPI(x0, M1, m1, f, eps):
    alpha = 2 / (M1 + m1)
    q = np.abs((M1 - m1) / (M1 + m1))
    x1 = x0 - alpha * f(x0)
    it = 1
    while abs(x1 - x0) > (1 - q) * eps / q:
        x0, x1 = x1, x1 - alpha * f(x1)
        it += 1
    print(f"–í—ã–ø–æ–ª–Ω–µ–Ω–æ {it} –∏—Ç–µ—Ä–∞—Ü–∏–π, x = {x1}")
    return x1
```
